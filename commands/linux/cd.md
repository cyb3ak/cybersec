## Change Directory - cd

#### Change to another directory

```text
cd dirpath
```

* Here `dirpath` can be absolute or relative

#### Change to the parent directory

```text
cd ..
```

#### change to the home directory

```text
cd ~
```

Go back to the previous directory \(back\)

```text
cd -
```
